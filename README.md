# AskMe
Basic Question/Answer user system clonce to ask.fm site

## Features
- Login
- Sign Up
- Print Questions To Me
- Print Questions From Me
- Answer Question
- Delete Question
- Ask Question
- List System Users
- Feed
- Logout
  
## Using
- C#
- OOP
- Collections
- Files
